## Project Overview

The Sales Insights Dashboard is an interactive Business Intelligence solution developed using Power BI. The dashboard provides a comprehensive analysis of sales performance, revenue trends, profit margins, customer behavior, and regional performance.

This project helps businesses make data-driven decisions by transforming raw sales data into meaningful visual insights.



## Objectives

- Analyze overall sales performance.
- Track revenue and profit trends over time.
- Identify top-performing products and categories.
- Understand customer purchasing patterns.
- Compare sales across different regions.
- Support business decision-making through interactive visualizations.

## 📂 Dataset Information

The dataset contains sales transaction records with the following fields:

Order ID -Unique order identifier , Order Date -Date of purchase ,Customer Name -Customer details ,Product Name - Product sold ,Category - Product category,  Quantity -Quantity purchased, Sales -Revenue generated ,Profit -Profit earned, Region - Sales region , State - Customer state .


## 🧹 Data Cleaning & Preparation

The following preprocessing steps were performed:

- Removed duplicate records.
- Handled missing values.
- Standardized region and category names.
- Converted columns to appropriate data types.
- Created calculated measures using DAX.
- Built relationships between tables.
